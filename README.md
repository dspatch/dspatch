Open-source agent orchestration platform.
Sandboxed Docker execution. Signal Protocol encryption. Cross-platform control.

---

- **Sandboxed execution** — Each agent runs in an isolated Docker container with its own daemon. Full root inside, completely sealed from the host.
- **End-to-end encrypted** — Signal Protocol (X3DH + Double Ratchet). The server relays ciphertext it cannot decrypt.
- **Multi-device orchestration** — Launch agents on any desktop from any device. All commands encrypted end-to-end.
- **Human-in-the-loop** — Agents escalate decisions with structured context and suggestions. Push notifications on every platform.
- **Python SDK** — One async function, one context object. Bring any model, any framework.
- **Open source core** — Full local orchestration, Docker sandbox, complete dashboard. No server required. SaaS tier adds multi-device sync.
